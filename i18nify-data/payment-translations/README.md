# Payment Translations Data

Canonical payment UI translation strings for 7 Indian languages.

## Schema

- **Root key**: `payment_translation_information`
- **Entry key**: BCP 47 language code (`hi`, `bn`, `mr`, `gu`, `ta`, `te`, `kn`)
- **Total entries**: 7 languages

## Supported Languages

| Code | Language | Native Name |
|------|----------|-------------|
| `hi` | Hindi | हिन्दी |
| `bn` | Bengali | বাংলা |
| `mr` | Marathi | मराठी |
| `gu` | Gujarati | ગુજરાતી |
| `ta` | Tamil | தமிழ் |
| `te` | Telugu | తెలుగు |
| `kn` | Kannada | ಕನ್ನಡ |
